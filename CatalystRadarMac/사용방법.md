# CRT Mac Beta 사용방법

이 앱은 실시간 매매 앱이 아니라, 지난 거래일의 실제 미국 주식 움직임을 조사하는 개인용 베타입니다.

직접 코드를 보고 수정하려면 [Xcode에서_수정하기.md](Xcode에서_수정하기.md)를 먼저 읽고 `CatalystRadar.xcodeproj`를 Xcode로 여세요.

## 앱 열기

1. `build` 폴더 안의 `CRT.app`을 더블클릭합니다.
2. macOS가 처음 실행을 확인하면 `열기`를 선택합니다.
3. 앱 오른쪽의 `설정` 버튼을 누릅니다.

다른 사람에게 시험판을 보낼 때는 `build` 폴더의 `CRT-Mac-Beta.zip` 파일을 전달하면 됩니다. 현재 파일은 개인 베타용 서명 상태이므로, 받는 사람이 처음 열 때는 앱을 마우스 오른쪽 버튼으로 누른 뒤 `열기`를 선택해야 할 수 있습니다.

## 가장 먼저 시험할 기능

처음에는 `전체시장 급변 후보 스캔`만 사용하면 됩니다.

1. [Massive](https://massive.com/)에서 무료 Stocks Basic 계정을 만듭니다.
2. 발급받은 API Key를 앱의 `Massive API Key` 칸에 입력합니다.
3. `SEC 조회용 이메일`에는 본인 이메일을 입력합니다. SEC 공식 공시 조회 시 자동 접근 주체를 알리기 위한 값입니다.
4. `저장`을 누릅니다. 입력한 정보는 Mac 키체인에 저장됩니다.
5. 메인 화면에서 지난 거래일 날짜를 선택합니다.
6. `전체시장 급변 후보 스캔` 버튼을 누릅니다.

앱은 전체 미국 종목의 지난 날짜 자료에서 움직임이 큰 후보를 찾은 다음, 상위 후보들의 1분봉과 SEC 공시 여부를 확인합니다.

## 뉴스까지 확인하기

관심 있는 종목들의 뉴스까지 보고 싶을 때 추가로 설정합니다.

1. [Alpaca](https://alpaca.markets/) 무료 계정을 만듭니다.
2. API Key와 Secret Key를 앱 설정에 입력하고 저장합니다.
3. 메인 화면의 `관심종목`에 티커를 입력합니다. 예: `IONQ, RGTI, QBTS`.
4. `관심종목 뉴스·공시 분석` 버튼을 누릅니다.

## 화면에 나타나는 뜻

- `공시 확인`: 선택 날짜에 SEC 공시가 확인된 후보입니다.
- `뉴스 확인`: 급변 시점 주변에 연결되는 뉴스가 확인된 후보입니다.
- `원인 미확인`: 확인된 공시나 시점 주변 뉴스가 없는 후보입니다.

`원인 미확인`은 매수 기회를 뜻하지 않습니다. 저유동성이나 단기 수급에 따른 움직임일 수 있습니다.

## 현재 한계

- 실시간으로 현재 급등 종목을 감시하지 않습니다.
- 전체시장 무료 스캔은 호출 한도 때문에 일별 후보 중 상위 4개만 분봉으로 자세히 확인합니다.
- 분봉의 저가와 고가로 변동 후보를 찾으므로, 실제로 그 가격에 거래할 수 있었다는 뜻은 아닙니다.
- 투자 추천, 자동매매, 목표가 제시는 하지 않습니다.

## 사람들에게 정식 배포하려면

현재 버전은 직접 시험하고 가까운 사용자에게 테스트를 부탁하는 단계에 적합합니다. 불특정 다수가 경고 없이 설치하는 형태로 배포하려면 Apple Developer Program 가입과 앱 서명·공증 절차가 필요하며, Apple 가입 비용이 발생합니다.
