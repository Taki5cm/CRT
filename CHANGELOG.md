# Changelog

이 파일은 버전마다 사용자에게 보이는 변경점과 현재 한계를 기록합니다. 되돌리기 기준점과 저장 데이터 호환성 원칙은 [Versioning and Rollback](docs/VERSIONING.md)을 참고하세요.

## CRT 0.11 (build 0.11.0)

Supporter ML이 나중에 학습할 수 있도록, 단순히 `300% 여부`만 저장하던 후보 검증을 사건 데이터셋 형태로 확장한 버전입니다.

- `supporter-training.sqlite`를 스키마 버전 `2`로 올리고 기존 후보를 보존한 채 학습용 열 추가
- 후보 검증 시 거래일 시가·종가·거래량·추정 거래대금·고점 시각·세션·고점 도달 시간을 함께 저장
- 고점 이후 `1`, `5`, `15`, `60`분 성과와 종가 기준 고점 대비 되돌림을 계산
- 검증 후보마다 `강세 유지형`, `되돌림 경계`, `고위험 급락형` 등 초기 위험 라벨 기록
- Supporter ML 화면에서 위험 라벨 수, 저장소 버전, 후보별 고점·거래대금·후속 성과를 표시
- Supporter ML 후보와 검증 결과를 CSV로 내보내는 기능 추가
- 앱 표시 이름, 빌드 스크립트, 알림 문구, README를 `CRT 0.11` 기준으로 갱신

### Deliberate Limits

- `0.11`의 위험 라벨은 학습 데이터 분류용 규칙 라벨이며 매매 확률이나 추천이 아닙니다.
- 뉴스·SEC 근거를 학습 행에 완전히 결합하는 작업은 다음 단계로 남겨두며, 현재는 가격 경로 중심의 구조화 저장을 우선합니다.
- 무료 IEX 분봉 기반 검증은 전체시장 확정 학습 데이터가 아니므로, 상용화 전 SIP 또는 라이선스가 확인된 역사 데이터 재검증이 필요합니다.

## CRT 0.10 (build 0.10.0)

실시간 감시를 차트 중심으로 읽을 수 있게 하고, 역사적 극단 급등 사례를 검증해 쌓는 Supporter ML 데이터 기반을 시작한 버전입니다.

- 감시 화면에 선택 종목의 캔들 차트 추가, 기본 `1분봉`과 `3분`, `5분`, `15분` 전환 지원
- 입력한 티커 조회, TOP 20·실시간 가격·포착 로그 클릭으로 차트 종목 즉시 전환
- Alpaca 과거 1분봉을 먼저 표시하고 감시 중 수신되는 해당 종목 체결을 현재 캔들에 실시간 반영
- 관심종목이 아닌 차트 조회 종목은 별도 구독하되 급등락 포착 조건 계산에는 포함하지 않도록 분리
- `supporter-training.sqlite` 로컬 학습 후보 저장소 추가 및 스키마 버전 `1` 기록
- `BIRD`, `ASTC`, `AIXI`를 최초 검증 대기 후보로 준비하고, 사용자가 티커·거래일을 추가할 수 있는 화면 제공
- Alpaca IEX 역사 분봉으로 직전 수신 거래일 종가 대비 장중 고가 `+300%` 충족 여부를 검증해 사건 또는 대조 표본으로 저장
- 표본 수, 확인된 300% 사건 수, 대조 표본 수를 화면에 표시하며 검증 전 예측 확률은 표시하지 않음

### Deliberate Limits

- 현재 후보 검증은 무료 시험이 가능한 IEX 분봉 기준입니다. 전체 미국 시장 학습 자료로 확정하려면 SIP 또는 라이선스가 확인된 전체시장 역사 데이터 검증이 필요합니다.
- 이번 버전은 사례 수집·검증 기반이며, 상승 기대률·하락 위험률 예측이나 자동 매매 기능을 제공하지 않습니다.

## CRT 0.9 (build 0.9.0)

뉴스·공시 연결이 실패하는 상황을 복구하고, 실시간 감시 제어와 조사 화면의 접근성을 높인 버전입니다.

- 전체시장 사후 후보에도 Alpaca 뉴스 조회를 연결해 공시 장애와 별개로 뉴스 근거 표시
- Massive Ticker Overview의 CIK를 우선 사용하여 SEC 종목 목록 `403`에 의존하지 않고 제출 이력 조회
- SEC 요청 식별 헤더를 명확히 하고, Massive CIK가 없는 경우 기존 SEC 종목 매핑을 예비 경로로 유지
- 포착 기록과 사후 결과 카드에서 `Stock Titan 최신 뉴스`와 `Google News` 교차 확인 링크 제공
- 재알림 제한을 `없음`, `10초`, `30초`, `1·2·5·15·30분`으로 확장하고, 선택값 저장 및 감시 중 즉시 반영
- 상태 칩과 강조 배경을 포함한 대시보드형 화면 개선
- 저장 데이터 구조는 `0.8`과 동일한 SQLite 스키마 버전 `2`를 유지

### Deliberate Limits

- Stock Titan 및 Google News는 사용자가 원문을 교차 확인하도록 여는 링크이며, 앱이 외부 페이지를 수집하거나 원인을 자동 확정하지 않습니다.
- SEC 제출 이력은 공개 공시 확인용이며, 희석 관련 양식 표시만으로 오퍼링 실행 여부를 단정하지 않습니다.

## CRT 0.8 (build 0.8.0)

포착된 급등락 후보에 뉴스·공시·기업 규모 근거를 자동으로 붙여, 단순 경보에서 2차 조사 보고로 확장한 버전입니다.

- 실시간 급등락 포착 직후 Alpaca 최신 뉴스 자동 조사
- SEC 최근 30일 공시를 확인하고 `S-1`, `S-3`, `424B`, `EFFECT` 등 희석 가능성 관련 양식을 별도 경고
- 설정에 Massive API Key가 있는 경우 시가총액, 해당 주식 종류 발행주식 수, 가중 발행주식 수 표시
- 포착 기록 카드에 뉴스 링크, 희석 관련 SEC 원문 링크, 2차 조사 상태 표시
- 예전 기록이나 부분 실패 기록을 위한 `2차 조사 실행`/`다시 조사` 동작 추가
- 1차 가격 경보 이후 조사 결과를 알리는 별도 Mac 알림 추가
- CSV 내보내기에 조사 상태, 기업 규모, 희석 양식, 뉴스·공시 항목 추가
- 로컬 SQLite 저장소를 스키마 버전 `2`로 마이그레이션하며 기존 0.7 기록을 보존

### Deliberate Limits

- SEC 희석 관련 양식이 있다는 표시는 실제 오퍼링 실행이나 매도 압력을 확정하는 판단이 아닙니다. 원문 확인을 위한 경고입니다.
- 정확한 최신 유통주식 수, 소유주주비율, short interest는 검증된 제공 권한 없이 표시하지 않습니다.
- 공개 일별 short sale volume은 실시간 공매도 포지션이 아니므로, 이번 버전의 초단위 포착 판단에는 섞지 않습니다.

## CRT 0.7 (build 0.7.0)

포착 순간만 보여주던 감시를, 이후 움직임을 계속 쌓아 검증 가능한 표본으로 만드는 첫 저장 버전입니다.

- 실시간 급등락 포착 이벤트를 사용자 Mac 내부 SQLite 데이터베이스에 자동 저장
- 티커, 방향, 세션, 피드, 감지 조건, 포착 가격과 거래대금을 기록
- 포착 후 같은 종목의 후속 체결을 따라 `1분`, `5분`, `15분` 변화율 기록
- 앱을 다시 열어 감시를 재개했을 때 진행 중인 종목의 후속 기록 연결 시도
- 포착 후 30분 안에 후속 체결을 충분히 받지 못한 기록을 `후속 수신 누락`으로 분리해 학습 자료 오염 방지
- 포착 이후 받은 체결 중 고점과 저점 변화를 함께 표시
- 최근 기록을 확인하는 `포착 기록과 사후 성적표` 화면 추가
- 전체 포착 기록을 CSV로 내보내 향후 검증·분석에 사용할 수 있도록 지원
- 데이터베이스 스키마 버전 `1`을 기록해 이후 마이그레이션과 롤백에 대비
- macOS 배포 아이콘의 고해상도 파일을 권장 크기에 맞춰 빌드 경고 제거

### Outcome Scope

후행 결과는 포착 뒤 각 시간을 지난 후 앱이 처음 받은 체결 가격으로 계산합니다. 앱이 실행 중이고 선택한 피드가 후속 체결을 실제로 수신해야 결과가 채워지며, 감시를 다시 시작한 경우에도 해당 종목이 현재 감시 범위에 포함되어야 합니다. 무료 IEX 표본은 전체시장 표본이 아닙니다.

### Supporter AI Scope

0.7은 학습 입력이 될 표본 수집을 시작하는 단계입니다. 기록이 충분히 쌓이고 시간 순서 검증을 통과하기 전까지 상승 기대률이나 하락 위험률을 예측 숫자로 표시하지 않습니다.

## CRT 0.6 (build 0.6.0)

실시간 감시를 급등뿐 아니라 급락까지 확장하고, 현재 수신 중인 종목의 변동 상황을 한눈에 살피는 버전입니다.

- 포착 방향을 `급등·급락`, `급등만`, `급락만` 중 선택
- 시간창 안 고점 대비 하락도 계산해 급락 로그와 Mac 알림 표시
- 현재 수신 스트림의 절대 변동률 기준 TOP 20 보드 추가
- 상승은 초록색, 하락은 빨간색으로 상태 및 로그 표시
- 좁았던 감시 조건 영역을 두 줄로 나누어 설정 가독성 개선
- 과거 학습형 위험·기대 분석 서포터의 준비 영역 추가

### Market Coverage

TOP 20은 앱이 실제 수신한 체결 범위의 순위입니다. 무료 IEX 관심종목 모드에서는 입력한 종목만 비교하며, 유료 SIP 전체시장 모드에서는 전체시장 수신을 기반으로 비교합니다. Alpaca의 별도 market movers 조회는 제공되지만 프리마켓 초단위 포착 목적과 동일한 자료로 가정하지 않습니다.

### Supporter AI Scope

하락 위험률과 상승 기대률은 충분한 과거 표본, 포착 후 관측 구간, 성능 검증 없이 표시하면 오해를 만들 수 있습니다. 이번 버전은 분석 영역과 필요한 입력 기준을 제시하며, 실제 검증된 확률 모델은 이후 데이터셋 구축 뒤 연결합니다.

## CRT 0.5 (build 0.5.0)

실시간 감시가 실제 체결 데이터를 받고 있는지 사용자가 직접 확인할 수 있게 하고, 입력 가독성을 개선한 점검 버전입니다.

- 감시 중 `수신 체결`, `마지막 수신`, `감시 범위` 상태 표시
- 무료 IEX에서 프리마켓 움직임이 누락될 수 있는 이유를 화면에 안내
- `수신 확인용 기준 적용` 버튼으로 저강도 움직임에서 연결 시험 가능
- 감시 시작 이후 수신된 체결만 급등 계산에 포함된다는 안내 추가
- 거래대금 입력에 천 단위 쉼표와 넓은 입력 폭 적용

### Diagnosis

`관심종목 · 무료 IEX`는 Alpaca 무료 계정의 단일 거래소 피드입니다. 화면에서 종목 가격이 움직이더라도 해당 움직임이 IEX 체결로 수신되지 않거나 감시 시작 전에 이미 발생했다면 경보가 나오지 않습니다. 전체시장 프리마켓 포착 검증에는 SIP 권한이 필요합니다.

## CRT 0.4 (build 0.4.0)

실시간 감시가 제품의 중심임을 분명히 하고, 사용자가 자신의 데이터 권한에 맞는 감시 범위를 선택하도록 화면과 동작을 다시 구성한 버전입니다.

- 첫 화면의 중심을 지속 실행되는 `시장 감시 시작/중지` 영역으로 변경
- `관심종목 · 무료 IEX`와 `전체시장 · 유료 SIP`를 명확한 감시 모드로 선택
- 감시 실행 상태와 경과 시간 표시
- 연결이 끊기면 3초 후 자동 재연결 시도
- 같은 종목의 재알림 제한을 `1분`, `5분`, `15분` 중 선택
- 과거 날짜 조회는 `사후 분석` 영역으로 분리하고 1회 분석임을 표시

### Scope

전체시장 실시간 감시는 사용자 계정에 SIP 권한이 있는 경우에만 작동합니다. 알림은 가격·거래대금 조건에 따른 후보 경보이며, 자동 뉴스 조사·투자 추천·자동매매는 아직 제공하지 않습니다.

## CRT 0.3 (build 0.3.0)

사용자가 본인의 Alpaca 데이터 연결로 자기 Mac에서 실시간 급등 감지를 시험하는 데스크톱 스캐너 베타입니다.

- 실시간 체결 스트림에서 `1초`, `2초`, `5초`, `30초`, `60초` 급등 조건 감지
- 상승률, 최소 주가, 시간창 거래대금 기준 조절
- 무료 `IEX` 관심종목 감지와 유료 `SIP` 전체 상장주 감지 모드 분리
- 감지 후보 화면 로그와 macOS 즉시 알림 추가
- 같은 종목의 반복 경보를 일정 시간 제한

### Scope

현재 실시간 알림은 가격·거래대금 조건을 충족한 후보를 빠르게 알리는 1차 경보입니다. 자동 뉴스·공시 조사와 테마 동반 상승 기반 2차 보고는 다음 개발 단계입니다. 매수 추천이나 자동매매 기능은 포함하지 않습니다.

## CRT 0.2 (build 0.2.0)

분석을 반복해서 사용하기 쉬운 데스크톱 경험으로 다듬은 버전입니다.

- 날짜 버튼을 누르면 보이는 큰 캘린더 선택 화면 추가
- `YYYY-MM-DD` 직접 날짜 입력과 이전 해·다음 해 이동 추가
- 직전 거래일, 1주 전, 1개월 전 빠른 선택 버튼 추가
- 주말 선택 시 직전 평일로 자동 조정
- 결과를 공시, 뉴스, 원인 미확인으로 필터링하는 기능 추가
- 설정에서 켤 수 있는 macOS 분석 완료 알림 추가
- 후보가 발견되면 알림에 대표 티커와 후보 수 표시
- Alpaca 무료 IEX 스트림 기반 관심종목 실시간 가격 미리보기 추가
- Alpaca 무료 Paper 계정 호환을 위해 관심종목 과거 분봉도 IEX 피드로 변경
- Massive 인증 실패 시 API Key를 확인할 수 있도록 안내 메시지 개선
- Alpaca 과거 분봉 권한 오류 안내 개선

### Notification Scope

현재 알림은 사용자가 직접 실행한 과거 분석이 완료되었을 때 표시됩니다. 실시간 가격 미리보기는 IEX 관심종목 체결을 보여주지만, 급등 감시 알림은 아직 포함되지 않습니다.

## CRT 0.1 (build 0.1.0)

초기 기능형 베타입니다.

- SwiftUI 기반 macOS 앱과 Xcode 프로젝트 제공
- 지난 거래일 전체시장 급변 후보 스캔 지원
- 관심종목 과거 분봉, 뉴스, SEC 공시 분석 지원
- Massive, Alpaca, SEC EDGAR 공식 인터페이스 연결
- API 키의 macOS Keychain 보관
- 기능 검증용 웹 프로토타입과 핵심 분석 테스트 포함

### Known Limits

- 실시간 전체시장 모니터링과 자동매매는 지원하지 않습니다.
- 무료 호출 범위에 맞춰 전체시장 상세 분봉 분석은 상위 후보 4개로 제한합니다.
- 개인 테스트용 빌드이며 정식 배포용 서명·공증은 아직 적용하지 않았습니다.
