# CRT 0.2 사용방법

이 앱은 실시간 매매 앱이 아니라, 지난 거래일의 실제 미국 주식 움직임을 조사하는 개인용 베타입니다. 관심종목의 현재 가격을 시험해 볼 수 있는 무료 실시간 미리보기도 포함되어 있습니다.

직접 코드를 보고 수정하려면 [Xcode에서_수정하기.md](Xcode에서_수정하기.md)를 먼저 읽고 `CRT.xcodeproj`를 Xcode로 여세요.

## 앱 열기

1. `build` 폴더 안의 `CRT.app`을 더블클릭합니다.
2. macOS가 처음 실행을 확인하면 `열기`를 선택합니다.
3. 앱 오른쪽의 `설정` 버튼을 누릅니다.

다른 사람에게 시험판을 보낼 때는 `build` 폴더의 `CRT-0.2.zip` 파일을 전달하면 됩니다. 현재 파일은 개인 베타용 서명 상태이므로, 받는 사람이 처음 열 때는 앱을 마우스 오른쪽 버튼으로 누른 뒤 `열기`를 선택해야 할 수 있습니다.

현재 표시 이름 `CRT 0.2`는 지난 거래일 분석 기능과 사용감을 함께 시험하는 베타입니다.

## 가장 먼저 시험할 기능

처음에는 `전체시장 급변 후보 스캔`만 사용하면 됩니다.

1. [Massive](https://massive.com/)에서 무료 Stocks Basic 계정을 만듭니다.
2. 발급받은 API Key를 앱의 `Massive API Key` 칸에 입력합니다.
3. `SEC 조회용 이메일`에는 본인 이메일을 입력합니다. SEC 공식 공시 조회 시 자동 접근 주체를 알리기 위한 값입니다.
4. `저장`을 누릅니다. 입력한 정보는 Mac 키체인에 저장됩니다.
5. 메인 화면에서 날짜 버튼을 누릅니다. `2026-05-22`처럼 직접 입력하거나 큰 달력, `이전 해`·`다음 해`, `직전 거래일`·`1주 전`·`1개월 전` 버튼을 사용할 수 있습니다.
6. `전체시장 급변 후보 스캔` 버튼을 누릅니다.

앱은 전체 미국 종목의 지난 날짜 자료에서 움직임이 큰 후보를 찾은 다음, 상위 후보들의 1분봉과 SEC 공시 여부를 확인합니다.

주말 날짜를 선택하면 앱이 직전 평일로 자동 조정합니다. 미국 공휴일이나 휴장일은 실제 데이터 조회 과정에서 확인됩니다.

`Massive API Key 인증에 실패했습니다`라는 메시지가 나오면 Massive Stocks에서 발급한 API Key인지 확인하고 설정에서 다시 저장하세요. Alpaca 키나 Massive 로그인 암호를 입력하면 연결되지 않습니다.

## 뉴스까지 확인하기

관심 있는 종목들의 뉴스까지 보고 싶을 때 추가로 설정합니다.

1. [Alpaca](https://alpaca.markets/) 무료 계정을 만듭니다.
2. API Key와 Secret Key를 앱 설정에 입력하고 저장합니다.
3. 메인 화면의 `관심종목`에 티커를 입력합니다. 예: `IONQ, RGTI, QBTS`.
4. `관심종목 뉴스·공시 분석` 버튼을 누릅니다.

관심종목 분석의 과거 분봉도 무료 Paper 계정에서 이용 가능한 IEX 자료를 사용합니다. `Alpaca 과거 분봉 ... 403`이 나타났다면, 이전 앱이 전체시장 피드를 요청하던 문제이므로 최신 `CRT 0.2` 빌드를 다시 실행해 시험하세요.

## 실시간 가격 미리보기

1. 설정에 Alpaca API Key와 Secret Key를 입력해 저장합니다.
2. `관심종목` 칸에 확인할 티커를 최대 30개까지 입력합니다.
3. `관심종목 실시간 가격 미리보기`의 `실시간 연결`을 누릅니다.
4. 가격 카드가 표시되면 IEX 피드에서 막 수신된 체결 가격입니다. 끝낼 때는 `중지`를 누릅니다.

무료 Alpaca 계정에서 CRT가 사용하는 가격 자료는 IEX 한 거래소 기준입니다. 미국 전체 거래소의 급등 종목을 정확히 실시간 탐지하는 단계에는 유료 전체시장 데이터가 필요합니다.

## 분석 완료 알림 켜기

1. 앱의 `설정`을 엽니다.
2. `분석이 끝나면 Mac 알림 받기`를 켭니다.
3. macOS가 알림 허용을 물으면 `허용`을 선택합니다.

알림은 직접 실행한 사후 분석이 끝났을 때만 표시됩니다. 아직 시장을 실시간으로 계속 감시하면서 급등 즉시 알려주는 기능은 아닙니다.

## 화면에 나타나는 뜻

- `공시 확인`: 선택 날짜에 SEC 공시가 확인된 후보입니다.
- `뉴스 확인`: 급변 시점 주변에 연결되는 뉴스가 확인된 후보입니다.
- `원인 미확인`: 확인된 공시나 시점 주변 뉴스가 없는 후보입니다.

`원인 미확인`은 매수 기회를 뜻하지 않습니다. 저유동성이나 단기 수급에 따른 움직임일 수 있습니다.

결과가 많을 때는 결과 위의 `전체`, `공시`, `뉴스`, `원인 미확인` 필터로 필요한 종류만 볼 수 있습니다.

## 현재 한계

- 현재가 미리보기는 가능하지만 실시간으로 전체시장의 급등 종목을 찾아 알림하지는 않습니다.
- 전체시장 무료 스캔은 호출 한도 때문에 일별 후보 중 상위 4개만 분봉으로 자세히 확인합니다.
- 분봉의 저가와 고가로 변동 후보를 찾으므로, 실제로 그 가격에 거래할 수 있었다는 뜻은 아닙니다.
- 투자 추천, 자동매매, 목표가 제시는 하지 않습니다.
