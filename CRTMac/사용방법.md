# CRT 0.6 사용방법

이 앱은 사용자 본인의 데이터 키로 작동하는 개인용 시장 감시 베타입니다. `시장 감시 시작`을 누르면 중지할 때까지 급등과 급락 후보를 계속 확인하고, 지난 거래일 조사는 별도의 사후 분석으로 실행합니다. 매수 추천이나 자동매매 앱은 아닙니다.

직접 코드를 보고 수정하려면 [Xcode에서_수정하기.md](Xcode에서_수정하기.md)를 먼저 읽고 `CRT.xcodeproj`를 Xcode로 여세요.

## 앱 열기

1. `build` 폴더 안의 `CRT.app`을 더블클릭합니다.
2. macOS가 처음 실행을 확인하면 `열기`를 선택합니다.
3. 앱 오른쪽의 `설정` 버튼을 누릅니다.

다른 사람에게 시험판을 보낼 때는 `build` 폴더의 `CRT-0.6.zip` 파일을 전달하면 됩니다. 현재 파일은 개인 베타용 서명 상태이므로, 받는 사람이 처음 열 때는 앱을 마우스 오른쪽 버튼으로 누른 뒤 `열기`를 선택해야 할 수 있습니다.

현재 표시 이름 `CRT 0.6`은 사용자 키 기반 지속 시장 감시를 시험하는 베타입니다.

## 가장 먼저 시험할 기능: 시장 감시

처음에는 무료 `관심종목 · 무료 IEX` 모드로 감시가 계속 작동하는지 확인합니다.

1. [Alpaca](https://alpaca.markets/)에서 API Key와 Secret Key를 발급받습니다.
2. 앱 설정에 두 키를 입력해 저장합니다.
3. 첫 화면의 감시 모드에서 `관심종목 · 무료 IEX`를 선택합니다.
4. 감시할 티커를 입력하고, 처음 수신 시험 시에는 `수신 확인용 기준 적용`을 누릅니다.
5. `시장 감시 시작`을 누릅니다. 실행 상태와 경과 시간이 보이면 감시가 계속 작동 중입니다.
6. `수신 체결` 숫자와 `마지막 수신` 시간이 변하면 앱이 해당 피드의 실제 거래를 받고 있는 것입니다.
7. `포착 방향`에서 `급등·급락`, `급등만`, `급락만`을 선택할 수 있습니다.
8. 조건을 충족하는 체결이 들어오면 로그와 Mac 알림에 후보가 표시됩니다. `감시 중지`를 누를 때까지 실행됩니다.

전체 종목을 감시하려면 사용자의 Alpaca 계정에 SIP 실시간 데이터 권한이 있어야 하며, `전체시장 · 유료 SIP` 모드를 선택합니다.

무료 IEX 모드는 미국 전체 시장 화면과 같은 범위를 보지 않습니다. 프리마켓에서 다른 서비스에 NCRA 상승이 보여도 `수신 체결`이 `0건`이면, 앱이 잘못 계산한 것이 아니라 무료 IEX 피드에서 그 체결을 받지 못한 상태입니다. 또한 앱을 켜기 전에 이미 발생한 상승은 실시간 감지 로그에 소급 표시하지 않습니다.

## 지난 날짜 사후 분석

1. [Massive](https://massive.com/)에서 무료 Stocks Basic API Key를 발급받아 앱 설정에 입력합니다.
2. `SEC 조회용 이메일`에 본인 이메일을 입력하고 저장합니다.
3. `사후 분석` 영역에서 날짜와 기준을 고릅니다.
4. `1회 분석: 전체시장 후보`를 누르면 지난 날짜 전체시장 후보를 한 번 조회합니다.
5. 주말 날짜를 선택하면 앱이 직전 평일로 자동 조정합니다.

`Massive API Key 인증에 실패했습니다`라는 메시지가 나오면 Massive Stocks에서 발급한 API Key인지 확인하고 설정에서 다시 저장하세요. Alpaca 키나 Massive 로그인 암호를 입력하면 연결되지 않습니다.

## 뉴스까지 확인하기

관심 있는 종목들의 뉴스까지 보고 싶을 때 추가로 설정합니다.

1. [Alpaca](https://alpaca.markets/) 무료 계정을 만듭니다.
2. API Key와 Secret Key를 앱 설정에 입력하고 저장합니다.
3. 메인 화면의 `관심종목`에 티커를 입력합니다. 예: `IONQ, RGTI, QBTS`.
4. `관심종목 뉴스·공시 분석` 버튼을 누릅니다.

관심종목 분석의 과거 분봉도 무료 Paper 계정에서 이용 가능한 IEX 자료를 사용합니다. `Alpaca 과거 분봉 ... 403`이 나타났다면 최신 `CRT 0.6` 앱을 다시 실행해 시험하세요.

## 실시간 급등락 감지

1. 설정에 Alpaca API Key와 Secret Key를 입력해 저장합니다.
2. 무료로 시험할 때는 `관심종목 · 무료 IEX`를 고르고 `감시할 관심종목`에 티커를 최대 30개까지 입력합니다.
3. `포착 방향`, `감지 시간`, `변동률`, `최소 주가`, `시간창 거래대금`을 설정합니다. 거래대금은 `10,000`처럼 천 단위 쉼표로 표시됩니다.
4. `시장 감시 시작`을 누릅니다. 조건을 충족하면 급등 또는 급락 로그와 Mac 알림에 후보가 표시됩니다.
5. 유료 Alpaca SIP 데이터 구독이 있는 경우 `전체시장 · 유료 SIP` 모드를 선택할 수 있습니다.

무료 Alpaca 계정에서 CRT가 사용하는 가격 자료는 IEX 한 거래소 기준입니다. 전체 상장주 실시간 감지에는 Alpaca의 SIP 데이터 구독이 필요합니다. 감시 중 연결이 끊기면 앱은 3초 뒤 자동 재연결을 시도합니다. 현재 경보는 뉴스 확인 전의 가격 움직임 알림입니다.

## 현재 수신 급등락 TOP 20

감시를 시작하고 체결이 들어오면 `현재 수신 급등락 TOP 20`이 표시됩니다. 선택한 감지 시간 안에서 상승 또는 하락 폭의 절댓값이 큰 순서입니다.

- `관심종목 · 무료 IEX`에서는 입력한 관심종목 가운데 수신된 종목의 순위입니다.
- `전체시장 · 유료 SIP`에서는 전체시장 스트림을 연결했을 때 전체 종목 후보 순위로 사용할 수 있습니다.
- TOP 20 보드는 상황 파악용이며, 실제 알림에는 설정한 변동률과 최소 거래대금 기준이 추가 적용됩니다.

## 위험·기대 분석 서포터

0.6 화면에는 포착 후 `1분`, `5분`, `15분` 결과와 뉴스·거래대금 자료를 바탕으로 향후 위험·기대 분석을 붙일 영역이 표시됩니다. 현재 버전은 검증되지 않은 상승 기대률이나 하락 위험률을 숫자로 제시하지 않습니다. 실제 분석 모델은 충분한 과거 포착 사례를 저장하고 성능을 확인한 뒤 연결해야 합니다.

## 분석 완료 알림 켜기

1. 앱의 `설정`을 엽니다.
2. `분석이 끝나면 Mac 알림 받기`를 켭니다.
3. macOS가 알림 허용을 물으면 `허용`을 선택합니다.

알림은 직접 실행한 사후 분석 완료와 실시간 급등 후보 감지에 표시됩니다. 실시간 알림은 가격 기반 1차 경보이며 원인 확정이나 매수 판단이 아닙니다.

## 화면에 나타나는 뜻

- `공시 확인`: 선택 날짜에 SEC 공시가 확인된 후보입니다.
- `뉴스 확인`: 급변 시점 주변에 연결되는 뉴스가 확인된 후보입니다.
- `원인 미확인`: 확인된 공시나 시점 주변 뉴스가 없는 후보입니다.

`원인 미확인`은 매수 기회를 뜻하지 않습니다. 저유동성이나 단기 수급에 따른 움직임일 수 있습니다.

결과가 많을 때는 결과 위의 `전체`, `공시`, `뉴스`, `원인 미확인` 필터로 필요한 종류만 볼 수 있습니다.

## 현재 한계

- 무료 IEX 모드는 관심종목 감지용이며 미국 전체시장 자료가 아닙니다.
- 유료 SIP 전체시장 모드는 사용자가 직접 취득한 데이터 권한으로 자기 Mac에서 작동하는 시험 기능입니다.
- 전체시장 무료 스캔은 호출 한도 때문에 일별 후보 중 상위 4개만 분봉으로 자세히 확인합니다.
- 분봉의 저가와 고가로 변동 후보를 찾으므로, 실제로 그 가격에 거래할 수 있었다는 뜻은 아닙니다.
- 투자 추천, 자동매매, 목표가 제시는 하지 않습니다.
