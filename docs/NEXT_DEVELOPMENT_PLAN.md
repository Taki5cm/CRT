# Next Development Plan

## Current Starting Point

`CRT 0.12`는 실시간/조회 종목 차트와 Supporter ML 근거 저장을 한 단계 올린 상태입니다. 차트는 `1`, `3`, `5`, `15`분봉과 `일봉`을 지원하고, 버튼·드래그·핀치로 좌우 이동과 확대/축소가 가능합니다. Supporter ML 후보 검증에는 뉴스 수, SEC 공시 수, 희석 관련 양식, 근거 요약이 함께 저장됩니다.

남은 정리 작업은 Git 권한이 허용되는 환경에서 `CRT 0.12` 변경분을 커밋·Push하고 `v0.12.0` 태그를 만드는 것입니다.

## Next Session Checklist

3. 발견된 오류는 기능을 늘리기 전에 `0.12.1` 수정 범위로 우선 처리합니다.

## CRT 0.11: Training Data Expansion

목표는 학습 가능한 사건 기록을 제대로 만드는 것입니다. 단순히 급등 티커 목록을 늘리는 단계가 아니라, 각 사건의 전후 가격 경로와 결과를 같은 구조로 저장합니다.

### Completed

- `supporter-training.sqlite`에 사건 상세 구조 추가: 세션, 기준가, 고가, 거래대금, 고점 도달 시간, 후속 성과
- `+300%` 급등 뒤 `1`, `5`, `15`, `60`분 및 장 마감 결과 라벨 저장
- 급등 후 큰 하락 또는 되돌림을 초기 위험 라벨로 분류
- `+300%` 미달 후보를 대조 표본으로 저장
- 데이터셋 화면에 위험 라벨 수와 저장소 버전 표시
- 후보 검증 기록의 CSV 내보내기 제공

### Completion Standard

- 사용자가 등록한 과거 사건 하나가 가격 경로와 사후 결과까지 한 기록으로 저장됨
- 성공 사례만 모으지 않고 위험 라벨과 대조 표본이 함께 생성됨
- 앱 종료 후 다시 열어도 데이터셋 기록이 유지됨

## CRT 0.12: Chart Expansion And Evidence Joining

목표는 가격 경로 데이터에 당시 뉴스·SEC 근거를 결합하고, 사용자가 분봉과 일봉을 한 화면에서 빠르게 넘겨볼 수 있게 만드는 것입니다.

### Completed

- 후보 검증 시 당시 뉴스와 SEC 공시를 같은 사건 행에 연결
- 희석·오퍼링 관련 SEC 양식과 이후 공시를 위험 라벨 근거로 보존
- Supporter ML 저장소를 스키마 `3`으로 확장하고 CSV 내보내기에 근거 필드 추가
- 차트 구간에 `일봉` 추가
- 차트 좌우 이동, 최근 구간 이동, 확대, 축소 버튼 추가
- 차트 드래그와 핀치 조작 지원
- 차트 하단 거래량 막대 표시

### Completion Standard

- 조회 종목 또는 감시 종목에서 분봉과 일봉을 전환해 볼 수 있음
- 차트 버튼과 손동작으로 과거 구간을 넘겨 보고 확대/축소할 수 있음
- Supporter ML 저장 행에 뉴스·SEC·희석 관련 근거 요약이 남음

## CRT 0.13: Validation And First Supporter Model

목표는 Supporter ML이 그럴듯한 말을 하는 단계로 넘어가기 전에, 실제 검증 리포트로 믿을 수 있는 범위를 먼저 정하는 것입니다.

### Work Items

- 시간 순서 기준 학습/검증 분리
- 상승 기대, 급락 위험, 오경보율을 따로 측정
- 표본 부족 및 데이터 누락 경고 표시
- 뉴스·SEC 근거 원문 링크 보존
- 비슷한 가격대·거래량 조건인데 급등하지 않은 종목을 대량 대조군으로 생성
- 데이터 품질 화면에 `상승 사건`, `위험 사건`, `대조군`, `자료 누락`을 더 명확히 분리
- 검증을 통과하지 못한 점수는 앱에 노출하지 않는 안전 장치 유지

### Completion Standard

- 시간 순서 검증 보고서가 앱 또는 내보내기 파일로 확인됨
- 검증 전 표본과 검증 후 표본이 섞이지 않음
- 사용자가 보는 점수는 성능 기준을 통과한 경우에만 제한적으로 표시됨

## Data Scale Roadmap

`100만 개 이상` 목표는 검증된 학습 행 기준으로 관리합니다.

| 단계 | 목표 |
| --- | --- |
| 초기 | 사용자가 찾은 대표 사례와 대조군으로 저장 구조 검증 |
| 확장 | 라이선스가 확인된 전체시장 역사 자료에서 극단 급등 사건 일괄 탐색 |
| 라벨링 | 가격 경로, 뉴스, SEC, 희석 가능성, 이후 결과를 결합 |
| 대규모 학습 | 사건 창과 대조 창으로 `100만+` 학습 행 생성 |
| 제품 표시 | 시간 순서 성능 검증과 오경보 검토를 통과한 결과만 표시 |

유료화나 전체시장 정확도를 목표로 하기 전에 아래 사항을 확인합니다.

## Product Principle

CRT는 급등을 예언하거나 매수 버튼을 대신 누르는 도구가 아니라, 빠르게 움직이는 종목을 놓치지 않고 근거와 위험을 더 빨리 확인하게 돕는 리서치 서포터로 발전시킵니다. 충분한 검증을 거치기 전에는 확률이나 추천처럼 오해될 수 있는 표현을 제품에 넣지 않습니다.
