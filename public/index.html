FUNCTIONAL BETA

실제 사후 추적 분석

Massive 무료 계정으로 전체시장 후보를 찾고, Alpaca 무료 계정으로 관심종목 뉴스까지 확인하며 SEC 당일 공시를 함께 연결합니다.

무료로 실행 가능 API 키는 이 컴퓨터의 분석 요청에만 쓰이며 저장하지 않습니다.

전체시장 급변 후보 스캔

Massive 무료 키 필요

지난 날짜의 미국 전체 종목을 확인한 뒤, 움직임이 큰 후보를 1분봉으로 다시 검사하고 SEC 공시를 연결합니다.

무료 호출 한도에 맞춰 전체 일별 후보 중 상위 4개를 분봉으로 검사합니다. 속보 뉴스 연결은 아래 관심종목 분석에서 제공합니다.

Massive 무료 키를 입력하면 실제 전체시장 사후 스캔을 실행할 수 있습니다.

관심종목 뉴스·공시 확인

Alpaca 무료 키 필요

직접 고른 종목의 실제 분봉을 검사하고, Alpaca 과거 뉴스와 SEC 공시를 함께 붙입니다.

최대 30개 입력 종목을 확인합니다. 분봉의 저가-고가 범위로 후보를 찾으므로 당시 진입 가능 가격을 보장하지 않습니다.

Alpaca 무료 키를 입력하면 뉴스 포함 사후 분석을 실행할 수 있습니다.
