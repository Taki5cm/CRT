# Data, Privacy and Limitations

## External Data Sources

| 서비스 | 사용 목적 | 앱이 전송하는 정보 |
| --- | --- | --- |
| Massive | 전체시장 과거 분석, 포착 종목 기업 규모 및 SEC 연결용 CIK 조회 | Massive API Key, 날짜, 티커 |
| Alpaca | IEX/SIP 실시간 감지, 선택 종목 분봉 차트, 역사 급등 후보 가격 검증 및 뉴스 확인 | Alpaca API Key/Secret, 날짜, 티커 또는 전체종목 구독 선택 |
| SEC EDGAR | 당일·최근 공시 및 희석 가능성 관련 양식 확인 | 앱 식별 문자열과 사용자가 설정한 연락 이메일, 조회 CIK |
| Stock Titan / Google News | 사용자가 누른 경우 외부 뉴스 교차 확인 | 브라우저에서 여는 티커 검색 링크 |

## Credential Handling

- macOS 앱에서 저장한 Massive 및 Alpaca 키는 macOS Keychain에 보관됩니다.
- 분석 완료 알림 사용 여부와 재알림 제한 값은 사용자 Mac의 로컬 설정에만 저장되며, 알림 허용은 macOS가 관리합니다.
- 실시간 포착 기록, 후행 가격 성적, 뉴스 제목·원문 URL·공시 양식·기업 규모 조회 결과는 사용자 Mac의 `Application Support/CRT/capture-history.sqlite`에 저장됩니다.
- Supporter ML의 후보 티커·거래일과 `+300%` 가격 검증 결과는 사용자 Mac의 `Application Support/CRT/supporter-training.sqlite`에 별도로 저장됩니다.
- `CSV 내보내기`에는 포착 시세·조건·후행 결과·2차 조사 요약이 포함되며 API 키는 포함되지 않습니다.
- 웹 프로토타입에서 입력한 키는 분석 요청을 보내는 동안만 사용하며 저장 기능을 두지 않았습니다.
- API 키, `.env` 파일, 앱 빌드 결과물은 Git 저장 대상에서 제외합니다.
- 실제 API 키를 이 저장소의 코드, Issue, 화면 캡처 또는 로그에 기록하지 마세요.

## Interpretation Limits

- `공시 확인`은 해당 날짜에 SEC 공시가 발견되었음을 뜻하며, 가격 변동의 인과관계를 증명하지 않습니다.
- `뉴스 확인`은 시점 주변 뉴스 후보가 발견되었음을 뜻하며, 변동 원인의 확정이 아닙니다.
- `원인 미확인`은 조사한 소스에서 직접 근거를 찾지 못했다는 뜻이며, 매수 기회나 이상 거래 판정이 아닙니다.
- 1분봉 분석은 분봉 범위 안의 고가와 저가를 이용하므로 실제 진입·청산 가능한 가격을 보장하지 않습니다.
- `CRT 0.10` 알림은 완료된 사후 분석, 실시간 급등락 포착, 포착 후 2차 조사 결과를 알리며 거래 지시나 원인 확정이 아닙니다.
- 차트의 분봉은 사용자의 Alpaca 피드 범위에서 표시되며 무료 IEX 차트는 미국 전체 거래소의 완전한 시세가 아닙니다.
- 무료 IEX 감지는 일부 거래소 체결만 사용하며, 미국 전체시장 급등 여부를 나타내지 않습니다.
- 실시간 TOP 20은 앱이 수신한 범위의 변동 순위일 뿐, 무료 IEX 모드에서 미국 전체시장 순위로 해석할 수 없습니다.
- 위험률·기대률 예측치는 검증된 모델이 마련되기 전에는 표시하지 않습니다.
- 포착 후 결과는 각 시간 기준을 지난 뒤 앱이 처음 수신한 체결로 기록되므로 정확히 해당 초의 시장가격이라는 뜻은 아닙니다.
- 포착 후 30분 안에 충분한 후속 체결을 받지 못하면 결과는 완료 표본이 아니라 `후속 수신 누락`으로 구분됩니다.
- `희석 가능성 관련` 표시는 SEC 공시 양식 분류이며 실제 신규 주식 발행 완료, 발행 물량, 가격 하락을 확정하지 않습니다.
- Stock Titan과 Google News 링크는 외부 확인을 돕는 이동 경로이며 앱이 해당 사이트의 내용을 자동 수집하거나 평가하지 않습니다.
- float, 소유주주비율, short interest는 이번 버전에서 제공하지 않습니다. 공개 short sale volume도 현재 포지션 잔고와 같은 개념이 아닙니다.
- 실시간 스트림은 사용자가 입력한 키로 앱에서 데이터 제공자에게 직접 연결되며, CRT 서버에 전달하거나 저장하지 않습니다.
- 감시 모드 선택은 사용자 Mac의 로컬 설정에 저장되며 API 키는 계속 Keychain에만 저장됩니다.

## Distribution Notes

이 저장소는 개인 연구 및 제품 검증을 위한 베타 코드입니다. 앱 자체를 판매하더라도 투자정보 제공 여부에 대한 국내 규제 검토가 필요합니다. 서버가 결과나 시세를 여러 사용자에게 제공하는 형태로 운영하기 전에는 시세·뉴스 데이터의 상업적 표시 및 재배포 권리, Apple 배포 서명, 개인정보 처리, 관련 금융 규제를 별도로 확인해야 합니다.
